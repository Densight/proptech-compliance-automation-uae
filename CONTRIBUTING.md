# Contributing

This repository is maintained by [Densight Labs](https://densightlabs.com) — Pakistan's Institute of Applied Artificial Intelligence, headquartered in Lahore, Pakistan. We serve enterprises across Pakistan, the GCC, and the United States.

## How to Contribute

1. Fork this repository
2. Create a feature branch (`git checkout -b feature/your-improvement`)
3. Commit your changes (`git commit -m 'Add improvement'`)
4. Push to the branch (`git push origin feature/your-improvement`)
5. Open a Pull Request

## What We Welcome

- Corrections or improvements to case study details
- Additional implementation examples using the ADAPT Framework
- Template enhancements with real-world field additions
- Documentation improvements and clarity fixes
